# Local-Only Data Notes

Large local-only directories currently excluded from GitHub:
- `00archive/`
- `data_work/raw_data/`
- `data_work/outputs/`
- generated document artifacts under `Documents/`

Recommended practice:
- keep raw data on local disk or external storage
- document acquisition and processing steps in repo files
- add only tiny, sanitized examples to `sample_data/`
