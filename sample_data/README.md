# TidalLanes Sample Data

This folder is reserved for small, shareable sample inputs and outputs that are safe to sync to GitHub.

What to put here:
- tiny CSV or GeoJSON examples used for testing
- one or two hand-checked rows that demonstrate schema
- lightweight expected outputs for smoke tests

What not to put here:
- raw production data
- large generated outputs
- archived files
- licensed or restricted datasets

Suggested structure:
- `inputs/`: minimal sample inputs
- `outputs/`: expected sample outputs
- `schemas/`: column definitions or field notes

Before adding a file:
1. Confirm it contains no sensitive or restricted data.
2. Keep individual files comfortably below GitHub's large-file thresholds.
3. Prefer anonymized and reduced examples.
